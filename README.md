### 소개
> 당신의 OTT 취향을 찾아주는 콘텐츠 추천 서비스

POPCO는 사용자가 자신만의 OTT 콘텐츠 취향을 발견하고, 개인화된 추천을 통해 새로운 콘텐츠 경험을 제공하는 서비스입니다.  
귀여운 메인 캐릭터 **'POPCO'** 와 함께 사용자의 평가 이력, 선호도, 그리고 독자적인 **페르소나 진단 시스템**을 활용하여 영화 및 시리즈 콘텐츠를 지능적으로 추천합니다.
